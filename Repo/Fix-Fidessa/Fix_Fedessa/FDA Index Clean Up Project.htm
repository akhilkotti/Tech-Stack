Project Background 

Some reporting tools on FDA are experience performance problems. The solution is to add some indexes to facilitate the query. Also, there are some redundant index which should be removed. In this project, we will test the all the sprocs used by reporting tools thoroughly so that the index change will not deteriorate the performance of reporting tools.
