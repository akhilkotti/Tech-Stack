IoC and DI
	Layers:
	- C - Controller
	- S - Service
	- R - Repositories
	
	- Inversion of Control(IoC)
		- IoC is a principle
		- Giving control of creation of objects to Spring framework
	- Dependency Injection
		- Its a design pattern
		- Its the actual implementation of IoC. In spring to achive IoC we use DI.
		- Spring Frameork does this dependency Injection.
		- This is achieved by 3 techniques
			- Constructor Injection
			- Setter Injection
			- Field Injection using annotation @AutoWired - Not Recomended
